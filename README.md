# AI_Dependency_Career_anxiety_students
## Project Overview
Power BI dashboard analyzing the impact of AI dependency on student career readiness, burnout, career clarity, and resume confidence using data from 15,000 students.
## Key Insights
- Dataset contains approximately 15,000 students
- Average AI dependency score 4.54
- Average career readiness score 6'00
- Average AI usage Hours: 2.01 hours per day
- High AI user experiences the highest burnout levels
- career clarity increases slightly with AI usage
- Resume confidence decreases among heavy AI Users
- Most students fall within the low and moderate AI usage categories.
##Tools Used
-Kpi Cards
- Ai usage Distribution Analysis
- Burnout Analysisby AI Usage level
- Career readiness and career clarity comparison
- Resume confidence Trend Analysis
- Interactive filters (Genders, Academic Streams, Urban/Rural)
## Tools Used
- Power BI
- Dax
- Data Cleaning
- Data visualization
- SQL
## Business Value
This project helps educators and institutions understand how AI dependency influences students careerd development,confidence, and readiness for future employment.
